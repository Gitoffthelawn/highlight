* Block/Allow List Instructions

The optional allow list and block list can be used to customize which URLs receive autonomous highlighting.

The lists are comprised of match patterns—used for specifying groups of URLs—with one pattern entered on each line.

Patterns on the allow list take precedence over patterns on the block list.

To apply autonomous highlighting only for URLs matching specific patterns, add <all_urls> to the block list, and also add the specific patterns to the allow list.

Global Highlighting

Click a highlighter to set the corresponding highlight level on every tab. This is applied once, upon clicking the highlighter.
